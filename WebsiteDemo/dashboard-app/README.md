## Website Demo ReadMe

## Website Overview 
- Show a demo website that creates, stores, and tracks Ollama LLM generated marketing campaigns using a configuration of settings and guidelines provided in the campaign builder tab

- The dashboard tab is static but is intended to track the amount and types of campaigns created, along with the cost and engagement metrics.

- The campaign queue tab shows a list of recently generated campaigns, some are static but ones that have been generated and sent over from the campaign builder you can see the individual subject lines and open the first email body to personally edit it 
## Folder Overview
- The dashboard, campaign queue, and campaign builder all have individual html and css styling files named after them

- All structrual website work is done in html, styling done in css, and functionality done in java

- Both charts on dashboard.html were created using the juptyer notebook file product-chart in the matplotlib library
